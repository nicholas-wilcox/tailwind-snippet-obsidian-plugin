## [0.1.0] - 2023-06-22

### Added

- Created the Unofficial TailwindCSS Obsidian Plugin, which uses TailwindCSS and PostCSS
  to generate a CSS snippet from a vault's Markdown files.
- Users can opt to enable TailwindCSS's Preflight styles.
